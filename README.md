Hi 👋 My name is Coleman Matey
==============================

Software Developer
------------------

I have always loved computers but I started to code in 2017. I started out with C++ but couldn't do anything useful with it so I picked up Python two months later. I loved its simplicity. I began to learn the Django framework but quickly realized I'd need some frontend skills. I picked up HTML and CSS but gave up on the latter because I felt like I wasn't creative enough. I began to learn JavaScript as well. Now that i have introduced myself to web development, i have began to ground my self in these frontend technologies including Sass and Gulp and soon move on to deepening my understanding of Python and C#.

* 🌍  I'm based in Ghana

* 🚀  I'm currently working on [My Portfolio Website](http://www.colemanmatey.com)
* 🧠  I'm learning Sass
* 🤝  I'm open to collaborating on Front-end projects
* ⚡  I think deeply about simple things.
